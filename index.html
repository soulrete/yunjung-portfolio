Do you need a website?

사용자 경험 개선에 집중하는 풀스택 웹 개발자 김윤중입니다.

프론트엔드부터 백엔드, 인프라까지 서비스의 흐름 전체를 이해하고,
실제로 사용자에게 가치가 닿는 기능을 빠르고 안정적으로 구현하는 것을 가장 중요하게 생각합니다.
React, Spring Boot, Docker 기반의 웹 서비스를 주로 개발하며
"어떻게 하면 더 단순하고 명확한 제품이 될까?"를 항상 고민합니다.
실제 운영되는 서비스 개발 경험을 바탕으로
문제를 분석하고 스스로 해결하는 능력을 갖춘 개발자로 성장해 나가고 있습니다.
